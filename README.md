# 3DRayTracer
A Ray Tracing rendering tool written in C#, following along with "*Ray Tracing in One Weekend*" by Peter Shirley, Trevor David Black, and Steve Hollasch.

### Features currently implemented:
* Primitives:
    * Sphere
* Materials
    * Diffuse
    * Glossy
    * Metallic
    * Transparent (Glass)
* In-Camera Effects:
    * Depth of Field
